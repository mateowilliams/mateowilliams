## Featured Projects

### 🔬 [waterbottle-vision-pipeline](https://github.com/mateowilliams/waterbottle-vision-pipeline)
End-to-end **computer vision** pipeline that classifies a water bottle's fill level (`full` / `half` / `overflow`) with a **ResNet18** (transfer learning). Covers the full MLOps cycle: EDA → training → **FastAPI** REST API → **PostgreSQL** persistence (Alembic migrations) → **drift monitoring (PSI)** → retraining, all containerized with Docker Compose.
`PyTorch · FastAPI · PostgreSQL · Alembic · Docker`

### 📈 [market-risk-monte-carlo](https://github.com/mateowilliams/market-risk-monte-carlo)
Financial risk simulation: prices modeled with **Geometric Brownian Motion** calibrated on log-returns, **Monte Carlo** simulation (1,000 paths × 252 trading days) and **95% VaR** estimation with visual scenario reporting.
`Python · NumPy · Pandas · Matplotlib`

### 🫀 [heart-disease-predictor](https://github.com/mateowilliams/heart-disease-predictor)
End-to-end ML + **Flask** app that estimates heart-disease risk. Leakage-safe **scikit-learn Pipeline** (`ColumnTransformer`), stratified split, **Stratified K-Fold CV**, threshold tuning, model packaging with `joblib`, served via a Flask web app + JSON API.
`scikit-learn · Flask · joblib`
